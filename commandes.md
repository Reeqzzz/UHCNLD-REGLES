# Commandes

## Commandes joueur

| Commande | Description |
|---|---|
| `/craft` | Ouvre le menu des recettes de craft spéciales (Boussole d'Hyrule, Réceptacle de Cœur). |
| `/positions` | Affiche les positions du Temple du Temps, des 4 donjons et des 5 marchands. |
| `/spec` | (En spectateur uniquement) ouvre le menu de sélection de vue. |

## Items en jeu

En plus des commandes, certaines actions se font directement via des objets dans la hotbar :

- **Choisir une Équipe** (lit, lobby uniquement).
- **Choisir sa Race** (livre enchanté, pendant la phase de sélection de race).
- **Règles de la partie** (livre, dernier slot de la hotbar), affiche les règles et les commandes utiles dans le chat.

## Commandes admin

Réservées aux administrateurs (`/uhcnld <sous-commande>`) :

| Sous-commande | Effet |
|---|---|
| `start` | Lance la partie. |
| `end` | Termine la partie en cours. |
| `team` | Gestion des équipes. |
| `race` | Gestion des races. |
| `tp` | Téléportation de debug (Triforce/donjon/marchand/Temple). |
| `status` | Affiche le statut de la partie. |
| `editmap` | Passe en mode édition de la map (créatif, lobby uniquement). |
| `forcepvp` | Force l'activation immédiate du PvP + ouverture des donjons. |
| `forceborder` | Force le déclenchement immédiat du rétrécissement de bordure. |
| `give` | Donne un objet custom à un joueur. |
| `rubis` | Donne des Rubis à un joueur. |
